## ⛰ 环境搭建

在开始之前，请确保你的开发环境中包括 Node.js 和 npm 包管理器。

Angular 需要 Node.js 版本 22.0.0 或更高版本。

要检查你的版本，请在终端/控制台窗口中运行 `node -v`

## 🥌 使用方法

1. 安装 yarn
```shell
npm install -g yarn
```

2. 安装依赖
```shell
yarn install
```

3. 修改服务端地址
```
proxy.conf.js → host
```

4. 启动
```shell
yarn start
```

## 🍭 打包编译

1. 修改 `angular.json` 打包路径：
   > `projects.architect.build.options.outputPath`

2. 执行打包命令
```shell
yarn run build
```
